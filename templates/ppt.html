Features

Chatbot Interaction

The Mock interview is conducted by the chatbot. The chatbot is AIML based (i.e it is a rule based chatbot) have more than 70 unique questions

Voice to Voice Interface

Your interview skills will improve with practise. Voice Interface allows the user to practise this. For speech synthesis , web speech api is utilized which is only available on google chrome and mozilla with limited functionality.

Realtime Emotion Analysis

Emotion Analysis is done using deep learning. MobileNet is utilized to train the model with images over 7 categories of different emotions.

Sentiment Analysis

Sentiment analysis is done using nltk



Text Analysis

Provides insights of the repeated words and frequency.

Data Visualization

The data is visualized using chartJS

Report Generation

At the end of the interview a report is generateed for the students

Client side emotion detection

The neural network is running on the browser instead of the server.
